This module removes the default "Apps" filter in the "Apps" menu, replacing it with a grouping by state.

**Table of contents**

.. contents::
   :local:

Use Cases / Context
===================

It's been calculated that on average, a user installing modules manually removes the standard "Applications" filter around 15799 times per year.

Usage
=====

Open app "Applications" > modules and applications are grouped by state.
